### Основные методы Object

| Метод | Назначение |
|-------|-----------|
| `toString()` | Строковое представление объекта |
| `equals(Object)` | Сравнение объектов на равенство |
| `hashCode()` | Хеш-код для хеш-таблиц |
| `getClass()` | Получить класс объекта |
| `clone()` | Создать копию объекта |
| `finalize()` | Вызывается перед сборкой мусора (устарел) |

Первые три — самые важные. Их часто нужно переопределять.
